# Contribuire a H2C Protocol

Linee guida per contribuire al protocollo H2C.

## Aree di Contributo

- **Specifica**: Miglioramenti alla grammatica, nuovi blocchi, regole operative
- **Tooling**: Parser, validatore, transpiler (Python, JavaScript, Rust, Go)
- **Documentazione**: Traduzioni, tutorial, esempi, benchmark
- **Integrazioni**: MCP, LangGraph, AutoGen, Semantic Kernel, CrewAI
- **Test**: Nuovi scenari, stress test, validazione cross-modello

## Processo

1. Aprire una Issue per discussione
2. Fork del repository
3. Branch: `feature/nome-descrittivo`
4. Commit con messaggi chiari
5. PR con descrizione e riferimenti alla SPEC

## Standard

- Grammatica: EBNF (ISO 14977)
- Documentazione: Markdown
- Codice: Python 3.11+ / TypeScript 5+ / Rust 2024
- Licenza: MIT
